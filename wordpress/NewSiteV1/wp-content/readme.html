Whitney Monuments WordPress Theme Read Me

This document describes how to install and use your WordPress theme.

Summary

  1. Content Management

    1. Custom Post Types

      • Monuments

        The content of the Gallery page is coming from this Monuments custom post type. Enter a title for the monument and upload an image via the Feature Image thumbnail. Choose category for the monument so it will be sorted corectly in the Gallery page.

    2. Page Templates

      • Default Template

        This is the template that will be used by all pages by default.

      • Home

        The home page is divided into three sections.

        The first section is the Tab Banner in which you cant enter text and upload an image.

        The second section is the Tab Gallery in which you can add infinite number of gallery items for the slider

        The third section is the Tab Testimonial in which you can enter text and upload an image.

      • Gallery

        This page is using the Monuments custom post type and its showing all monuments and all monuments categories with which the user will be able to filter them.

    3. Theme Options

      Go to Theme Options » General and setup theme options

    4. Shortcodes

      • [year]

        Use this shortcode to display the current year.

      • [button link="#" text=""]

        Use this shortcode to display button. Example:

        [button link="#" text="Contact Us"
